# 亚马逊棋（Amazon Chess）—— 人机对战游戏

本项目是数算B大作业，实现了一个完整的 **亚马逊棋**（Amazon Chess）桌面游戏，支持人类玩家与 AI 对战，并提供三种难度的 AI 对手、悔棋、存读档、提示等功能。项目采用 Python + Tkinter 构建图形界面，AI 核心基于 **Minimax 搜索 + Alpha-Beta 剪枝** 与启发式评估函数，支持迭代深化。



本项目原为25秋季计算概论A大作业，但因我对于数算大作业没想到什么好的idea，于是将上学期的项目搬来。不过，上学期我所做的是基于c++的，在命令行进行文本输入来操作的版本，这次用python实现了图形化操作，并加入了不同难度的AI，优化成了一个更像游戏的游戏。

## 游戏介绍

亚马逊棋（又称“皇后棋”）是一款双人抽象策略棋类游戏，在 10×10 棋盘上进行。  

- 每方拥有 **4 枚亚马逊棋子**（人类：白色，AI：黑色）。  
- 亚马逊棋子的移动方式类似国际象棋的 **皇后**（横向、纵向、对角线任意距离），但不能越过其他棋子或障碍。  
- 每回合分为 **两步**：  
  1. 选择一枚己方棋子，移动到空位；  
  2. 在移动后的位置放置一个 **永久障碍**（灰色方块）。  
- 障碍会阻塞路径，且不可移动。  
- 当一方无法移动任何己方棋子时，该方失败，游戏结束。

## 核心算法

AI 的实现根据难度分为三个等级：

| 难度     | 算法                             | 说明                                     |
| ------ | ------------------------------ | -------------------------------------- |
| **简单** | 纯随机选择                          | 从所有合法走法中随机选取一步，无任何策略。                  |
| **中等** | 启发式随机                          | 优先移动到棋盘中心区域，并尽量将障碍放置在靠近人类棋子的位置（贪心策略）。  |
| **困难** | Minimax + Alpha-Beta 剪枝 + 迭代深化 | 使用启发式评估函数，搜索深度动态增加（2~4层），并限制分支因子以提高效率。 |

### 启发式评估函数

评估函数综合以下四项指标（加权求和）：

| 权重  | 指标    | 描述                              |
| --- | ----- | ------------------------------- |
| 40% | 移动自由度 | AI 所有棋子可走位置数之和 减去 人类棋子可走位置数之和。  |
| 30% | 战略控制  | 棋盘中心区域（4×4 内）的棋子价值，离中心越近加分越高。   |
| 20% | 威胁与安全 | 若某棋子可走位置 ≤3，则判定为受困，根据差值给予惩罚或奖励。 |
| 10% | 发展潜力  | 棋子周围 8 方向上的空位数量，反映未来活动空间。       |

最终得分 = 0.4×自由度 + 0.3×中心控制 + 0.2×威胁安全 + 0.1×发展潜力。

### 搜索优化

- **迭代深化**：从深度 2 开始，每次增加 1，直到深度 4 或超时（3 秒）。  
- **Alpha-Beta 剪枝**：配合启发式排序（对走法按评估值预排序），显著减少搜索节点。  
- **分支因子限制**：根节点最多考虑 20 个走法，后续节点最多 15 个，平衡效率与棋力。  
- **时间控制**：每次搜索最多耗时 3 秒，保证响应速度。

## 运行指南

### 环境要求

- Python 3.6 或更高版本
- 标准库：`tkinter`（一般随 Python 预装）、`json`、`random`、`math`、`time`、`os`、`sys`

### 安装与运行

1. **克隆或下载本项目**

2. **在vscode/pycharm中点击运行**

3. **游戏窗口将自动打开**，即可开始对战。

### 游戏操作说明

- **鼠标点击**：点击白色棋子（人类）选中，再依次点击浅蓝色（移动目标）和金黄色（障碍位置）完成回合。

- **取消选中**：点击已选中的棋子可取消选择。

- **控制按钮**：
  
  - `新游戏`：重置棋盘。
  - `保存/加载`：将当前游戏状态保存为 JSON 文件或从文件恢复。
  - `AI走棋`：强制 AI 立即行动（通常 AI 会自动响应）。
  - `悔棋`：撤销上一步（包括人类和 AI 的走法）。

- **菜单栏**：
  
  - `设置` → `AI难度`：可随时调整 AI 等级（1~3）。
  - `帮助` → `游戏规则` / `操作说明`：查看详细规则和界面指引。

### 游戏状态显示

- 顶部状态栏显示当前轮到谁，以及当前操作阶段（选择棋子、移动、放置障碍）。
- 右侧信息框实时记录每一步的走法坐标及 AI 搜索信息（困难难度）。
